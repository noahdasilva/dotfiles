# dotfiles
My personal dotfiles for tmux, zsh, &amp; whatever else I break.

## Contents

- [ssh](./ssh/README.md) — SSH keys and host config
- [tmux](./tmux/README.md) — tmux config with catppuccin theme
- [zsh](./zsh/README.md) — Oh My Zsh + Powerlevel10k

## Prerequisites

- [Homebrew](https://brew.sh/) (for MacOS)
- [Oh My Zsh](https://ohmyz.sh/)
- [Powerlevel10k](https://github.com/romkatv/powerlevel10k)
- [tpm](https://github.com/tmux-plugins/tpm)

## Install
```bash
git clone git@github.com:noahdasilva/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```
